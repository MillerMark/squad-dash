# Options

The **Options** window (accessible from the main menu or toolbar) lets you configure SquadDash to fit your workflow. It is divided into tabs — one per area of the application.

![Screenshot: Options window open on the General tab](images/options-overview.png)
> 📸 *Screenshot needed: The full Options/Preferences window with the tab strip visible on the left or top — General tab selected, showing the user name field.*

---

## Tabs at a Glance

| Tab | What you configure |
|---|---|
| [General](general.md) | Your display name in transcripts |
| [Speech](speech.md) | Voice-input provider, API keys, and voice text replacement rules |
| [Remote Access](remote-access.md) | Tunnel provider and auth token for public access |
| [Custom Model](custom-model.md) | Bring-your-own model (BYOK) — connect a local or third-party LLM |
| [Notifications](notifications.md) | Push notifications via ntfy when events occur |
| [Sounds](sounds.md) | Per-event audio cues and text-to-speech configuration |
| [AI](ai.md) | AI-assisted editor shortcuts |

---

## Opening Options

Go to **Edit → Preferences** (or the gear icon in the toolbar). Changes on most tabs are saved immediately — there is no separate Save button except where noted.
