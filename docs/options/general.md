# General

The General tab contains basic identity settings for your SquadDash session.

![Screenshot: General tab showing the User Name field](images/options-general.png)
> 📸 *Screenshot needed: The Options window with the General tab selected — show the User Name text box with a sample name entered.*

---

## Settings

### User Name

Your name as it appears in the **Transcript**, prefixed before every prompt you send.

- Set this to whatever you'd like the transcript to show — your real name, a handle, or a role label.
- The name is stored locally and never sent to any remote service on its own.
