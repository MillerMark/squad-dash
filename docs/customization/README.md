# Appearance

SquadDash gives you precise control over how the UI looks — base theme, color tint, accent, saturation, contrast, and font scale — all saved per workspace.

| Setting | Where |
|---|---|
| Light / Dark theme | View → Theme |
| Color tint (8 stops) | View → Tint |
| Accent color offset | Right-click active agent panel border |
| Contrast | View → Contrast |
| Saturation | View → Saturation |
| UI font scale | Ctrl+Scroll over title bar |

All settings except font scale are saved per workspace.
